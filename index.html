CSS(Cascading Style Sheets)

Cascading Style Sheets se traduz Folhas de Estilo em Cascata

CSS tem por objetivo estilizar o conteúdo de documentos HTML

HTML 5-Conteúdo

  • texto;
  • tabela;
  • imagem;
  • vídeo.

CSS 3-Estilização de Conteúdo

  • cores;
  • fontes;
  • tamanho;
  • sombras;
  • posicionamento

Sintaxe

Seletor{

propriedades:valor;/"comentário"/

propriedades-subpropriedades:valor;

}

p{

color:blue;

text-align:center

}

Formas de utilizar o CSS

INLINE
os códigos CSS são deifinidos dentro de uma tag HTML.Afetam somente a tag em que foi definida.
INCORPORADO
os códigos CSS são deifinidos no cabeçalho, dentro da tag HEAD.Afetam somente o documento em que foi definido.
EXTERNO
os códigos CSS são deifinidos em um arquivo externo.Afetam todos os documentos linkados ao CSS
